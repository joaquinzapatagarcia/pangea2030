# Material bruto

Zona de entrada para contenidos todavía no incorporados al canon.

Subdirectorios previstos:

- `textos-propios/`
- `diario/`
- `obras-arte/`
- `libros-lecturas/`
- `ideas-negocio/`
- `audio/`
- `experimental/`

Convención recomendada: `AAAA-MM-DD_tipo_nombre-descriptivo.ext`.

Guardar aquí un material no significa que ya forme parte del universo público.
