# Canon PANGEA 2030

Este directorio contiene únicamente materiales y decisiones aceptados como parte oficial del universo.

## Criterio de entrada

Un material entra en el canon cuando:

- amplía el mundo sin contradecir sus reglas vigentes;
- aporta lenguaje, memoria, personaje, territorio, señal o conflicto;
- puede rastrearse hasta una fuente;
- ha sido aprobado expresamente para PANGEA 2030.

Los borradores y materiales pendientes permanecen en `bruto/`.
